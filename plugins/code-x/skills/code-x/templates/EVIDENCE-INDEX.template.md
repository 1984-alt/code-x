# EVIDENCE INDEX — <project>

> One place that lists WHERE the proof lives for each module/card (screenshots, test logs, demo recordings). Kept light — `cx check evidence` reads it; `cx check final-ready` rolls it into the certificate. Load-on-demand.

**applicable:** yes (every real build)

| Card | Acceptance item | Evidence path | Type | Verified? |
|---|---|---|---|---|
| BUILD-001 | home screen renders | evidence/build-001/home.png | screenshot | ✅ |
| BUILD-001 | tests pass | evidence/build-001/test.log | test-log | ✅ |

*Rule: no acceptance item is "done" without a real, non-empty evidence path here. "Should work" is banned (Kernel rule 5).*
